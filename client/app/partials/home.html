DEVELOPER

The development of a new process is usually a long task, that needs time and debug, that are typically done on the developer PC. Once the processor is tested and ready, the user should be able to drag and drop it in WASDI to let this processor been automatic deployed in the cloud.
The technology is based on a smart multi language library (IDL, pyhon, Matlab, Ocave, Java) that basically simulates a virtual file system between the cloud and the developer PC, with very few and simple primitives.
The user can develop using this library in his own environment, and the WASDI code takes care of accessing missing data from the cloud once is needed, during the development and testing phase. Once the processor is ready and deployed, the system recognizes that the software is running on the cloud and makes direct access to the input data in a transparent way.

WASDI key feature is the possibility for EO Experts to develop their own processor and deploy it to the cloud with a simple drag and drop operation
